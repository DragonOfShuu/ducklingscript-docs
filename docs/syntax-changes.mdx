# Syntax Changes
As we mentioned in our first section Rubber Ducky 1.0 tends to have you quacking the same line over and over to get the results you want, we aim to change that and allow performing multiple actions with single commands. You can take a look at the new changes below!

<Tabs>
    <TabItem value="string_example" label="STRING" default>
    Rubber Ducky 1.0
    ```
    STRING hello
    STRING world
    ```

    DucklingScript
    ```
    STRING
        hello
        world
    ```
    </TabItem>


    <TabItem value="stringln_example" label="STRINGLN">
    Rubber Ducky 1.0
    ```
    STRINGLN Hello,
    STRINGLN world,
    STRINGLN it is I!
    ```

    DucklingScript
    ```
    STRINGLN 
        Hello,
        world,
        it is I!
    ```
    </TabItem>

    <TabItem value="var_example" label="VAR">
    DucklingScript
    ```
    VAR
        a 10
        b 30
        c "woah"
    ```
    </TabItem>


    <TabItem value="triple_quote_example" label="Triple Quotes">
    Example
    ```
    STRINGLN
        THIS
            WILL
                ERROR
    ```
    ```
    STRINGLN
        """
        THIS
            WILL
                NOT
                    ERROR
        """
    ```
    </TabItem>
</Tabs>
