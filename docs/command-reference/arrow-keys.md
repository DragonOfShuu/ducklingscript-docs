# ARROWKEYS
Emulates an input as if the user had pressed that arrow key.

**Syntax:** `<arrowDirection>`

Example
```
UP
DOWN
LEFT
RIGHT
UPARROW
DOWNARROW
LEFTARROW
RIGHTARROW
```

Action Log
```
Pressed UP
Pressed DOWN
Pressed LEFT
Pressed RIGHT
Pressed UP
Pressed DOWN
Pressed LEFT
Pressed RIGHT
```
