# SYSRQ [[Flipper Command]](https://developer.flipper.net/flipperzero/doxygen/badusb_file_format.html#autotoc_md72)
`SYSRQ` emulates an input as if the user had pressed the Linux Magic `SYSRQ` key.

**Syntax:** `SYSRQ`

Example
```
SYSRQ
```

Action Log
```
Pressed SYSRQ
```
