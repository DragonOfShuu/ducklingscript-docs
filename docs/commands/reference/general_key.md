General Keys are a list of different keys that you can use with Quackinter only.

:::note
General Keys are `Quackinter` only, meaning you can only use it along side with the built-in interpreter for DucklingScript; it is not supported for use on the flipper or generic Rubber Ducky Scripting Language 1.0.
:::

## Syntax

```
<PrimaryKey>+<SecondaryKey>+<SecondaryKey>+...
```

General keys are keys that can be separated using `+` symbols; or even used on their own.
## Example

DucklingScript
```
CTRL+SHIFT+ALT+DELETE
```
Action Log
```
HOLD CTRL
HOLD SHIFT
HOLD ALT
PRESS DELETE
RELEASE ALT
RELEASE SHIFT
RELEASE CTRL
```
