# GUI

`GUI`/`WINDOWS`/`META` emulates an input as if the user had pressed the windows/meta key and the `<chr>` if given.

## Syntax
```
<GUI|WINDOWS|META> <optional chr>
```

## Example:
```
GUI
GUI a
```
Output
```
Pressed WINDOWS/META
Pressed WINDOWS/META+a
```
