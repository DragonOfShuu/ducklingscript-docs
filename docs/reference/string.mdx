# STRING/STRINGLN
`STRING` emulates the user typing the given `<string>`. After the command and one space, all subsequent characters will be typed out.

`STRINGLN` emulates the user typing the given `<string>` and pressing `enter`.

Example
```
STRING Hello?
STRINGLN Hello!
STRING Goodbye!
```
